# Phase 1.9D: Exam Prep — Design Document

**Date:** 2026-03-05
**Scope:** Exam prep configuration UI, full mock mode, predictive scoring, backward planning, tests
**Approach:** Layered integration onto existing practice engine

---

## Decisions

1. **Exam dates** → New `exam_goals` table (not JSONB enrichment)
2. **Mock selection** → Both entry points: assessment_type filter on configure page + dedicated /practice/exam-prep page for full mocks
3. **Predictive scoring** → Simple percentage-to-grade-boundary mapping using existing GradingScale model
4. **Daily sessions** → Dashboard card + ExamPrepService auto-plan

---

## Service Layer

### ExamPrepService

```
App\Services\ExamPrepService

getActiveGoals(User $user): Collection<ExamGoal>
  → Active goals with assessmentType + gradingScale eager-loaded

getDailyPlan(User $user, ExamGoal $goal): array
  → Backward planning: days_remaining, topics_per_day, today_topics (weak first, then high-weight)
  → Output: { goal, days_remaining, topics_remaining, topics_per_day, today_topics[], suggested_question_count, suggested_time_minutes }

getAvailablePapers(AssessmentType $type, ?InstitutionCourse $course): Collection<QuestionPaper>
  → Published papers for assessment type, optionally filtered by course

createMockSession(User $user, QuestionPaper $paper): PracticeSession
  → mode=FullMock, question_paper_id, assessment_type_id, time_limit from paper.duration_minutes
  → Questions loaded in section sort_order → question sort_order

getPredictiveScore(PracticeSession $session): array
  → Maps score_percentage to grading scale boundaries
  → Output: { raw_score, percentage, grade, grade_label, pass_threshold, is_passing, next_grade, points_to_next }

getTopicGaps(User $user, ExamGoal $goal): Collection
  → Topics with <70% accuracy or zero attempts, ordered by gap severity
```

### New Types (resources/js/types/practice.ts additions)

```typescript
interface ExamGoal {
    id: string;
    assessment_type: { id: string; name: string; is_exit_exam: boolean; is_entrance_exam: boolean };
    institution_course: { id: string; course_code: string; course_title: string } | null;
    exam_date: string | null;
    target_score: number | null;
    is_active: boolean;
}

interface DailyPlan {
    goal: ExamGoal;
    days_remaining: number;
    topics_remaining: number;
    topics_per_day: number;
    today_topics: { id: string; title: string; accuracy: number; weight: number }[];
    suggested_question_count: number;
    suggested_time_minutes: number;
}

interface PredictiveScore {
    raw_score: number;
    percentage: number;
    grade: string;
    grade_label: string;
    pass_threshold: number;
    is_passing: boolean;
    next_grade: string | null;
    points_to_next: number | null;
}

interface SectionBreakdown {
    section_label: string;
    correct: number;
    total: number;
    marks_earned: number;
    marks_possible: number;
}

interface MockPaper {
    id: string;
    title: string;
    year: number | null;
    duration_minutes: number;
    total_marks: number;
    question_count: number;
    assessment_type: { id: string; name: string };
}
```
