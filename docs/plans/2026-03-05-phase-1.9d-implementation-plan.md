**Step 3: Write ExamPrepService implementation**

File: `app/Services/ExamPrepService.php`

Key methods and their implementations:

**`getActiveGoals(User $user): Collection`** — Query `ExamGoal::where('user_id', $user->id)->where('is_active', true)` with eager-loaded `assessmentType.gradingScale` and `institutionCourse`.

**`getAvailablePapers(AssessmentType $type, ?InstitutionCourse $course = null): Collection`** — Query `QuestionPaper::where('assessment_type_id', $type->id)->published()`. When `$course` provided, also filter by `institution_course_id`. Select: id, title, year, duration_minutes, total_marks, assessment_type_id, institution_course_id. Add `withCount('questions')`.

**`createMockSession(User $user, QuestionPaper $paper): PracticeSession`**:
1. Load `$paper->questions()->orderBy('question_section_id')->orderBy('sort_order')->get()`
2. Create PracticeSession with: `mode = PracticeMode::FullMock`, `question_paper_id = $paper->id`, `assessment_type_id = $paper->assessment_type_id`, `institution_course_id = $paper->institution_course_id`, `time_limit_seconds = $paper->duration_minutes * 60`, `question_ids = $questions->pluck('id')`, `question_count = $questions->count()`, `is_resumable = true`, `last_activity_at = now()`

**`getPredictiveScore(PracticeSession $session): ?array`**:
1. If no `assessment_type_id`, return null
2. Load `$session->assessmentType->gradingScale`
3. If no grading scale, return null
4. Get `$percentage = $session->score_percentage`
5. Walk `grade_boundaries` array — find boundary where `$percentage >= boundary['min'] && $percentage <= boundary['max']`
6. Find next higher grade (boundary with next higher `min`)
7. Return: `['percentage' => $percentage, 'grade' => $boundary['label'], 'is_passing' => $boundary['is_pass'], 'pass_threshold' => $scale->pass_threshold, 'next_grade' => $nextBoundary['label'] ?? null, 'points_to_next' => $nextBoundary ? ($nextBoundary['min'] - $percentage) : null]`

**`getTopicGaps(User $user, ExamGoal $goal): Collection`**:
1. Query `PracticeAnswer` grouped by topic (via `question.topicLinks.canonical_topic`)
2. Filter: `user_id` matches, `is_correct IS NOT NULL`, `institution_course_id` matches goal's course (if set)
3. Group by topic, calculate accuracy = `correct / total * 100`
4. Return topics with accuracy < 70% OR zero attempts, ordered by accuracy ASC

**`getDailyPlan(User $user, ExamGoal $goal): ?array`**:
1. If no `exam_date`, return null
2. `$daysRemaining = max(1, now()->startOfDay()->diffInDays($goal->exam_date))`
3. Get all topics for the goal's course via `CourseTopicMapping`
4. Get per-topic accuracy from practice_answers
5. Sort: weak topics first (< 70%), then by weight DESC
6. `$topicsPerDay = ceil(remaining_topics / $daysRemaining)`
7. Slice today's topics
8. Return: `['goal' => ..., 'days_remaining' => ..., 'topics_remaining' => ..., 'topics_per_day' => ..., 'today_topics' => [...], 'suggested_question_count' => min(20, available), 'suggested_time_minutes' => ...]`

```php
<?php

namespace App\Http\Controllers\Student;

use App\Http\Controllers\Controller;
use App\Http\Requests\Student\StartMockExamRequest;
use App\Models\QuestionPaper;
use App\Services\ExamPrepService;
use Illuminate\Http\JsonResponse;
use Illuminate\Http\RedirectResponse;
use Illuminate\Http\Request;
use Inertia\Inertia;
use Inertia\Response;

class ExamPrepController extends Controller
{
    public function __construct(private ExamPrepService $examPrepService) {}

    public function index(Request $request): Response|RedirectResponse
    {
        $user = $request->user();
        $goals = $this->examPrepService->getActiveGoals($user);

        if ($goals->isEmpty()) {
            return redirect()->route('practice.configure')
                ->with('info', 'Set up your exam goals to unlock Exam Prep.');
        }

        $papers = [];
        foreach ($goals as $goal) {
            $goalPapers = $this->examPrepService->getAvailablePapers(
                $goal->assessmentType,
                $goal->institutionCourse
            );
            $papers[$goal->id] = $goalPapers->map(fn ($p) => [
                'id' => $p->id,
                'title' => $p->title,
                'year' => $p->year,
                'duration_minutes' => $p->duration_minutes,
                'total_marks' => $p->total_marks,
                'question_count' => $p->questions_count,
            ]);
        }

        return Inertia::render('practice/exam-prep', [
            'goals' => $goals->map(fn ($g) => [
                'id' => $g->id,
                'assessment_type' => [
                    'id' => $g->assessmentType->id,
                    'name' => $g->assessmentType->name,
                    'is_exit_exam' => $g->assessmentType->is_exit_exam,
                    'is_entrance_exam' => $g->assessmentType->is_entrance_exam,
                ],
                'institution_course' => $g->institutionCourse ? [
                    'id' => $g->institutionCourse->id,
                    'course_code' => $g->institutionCourse->course_code,
                    'course_title' => $g->institutionCourse->course_title,
                ] : null,
                'exam_date' => $g->exam_date?->toDateString(),
                'target_score' => $g->target_score,
                'days_remaining' => $g->exam_date ? max(0, now()->startOfDay()->diffInDays($g->exam_date, false)) : null,
            ]),
            'papers' => $papers,
        ]);
    }

    public function start(StartMockExamRequest $request): RedirectResponse
    {
        $user = $request->user();
        $paper = QuestionPaper::findOrFail($request->validated('question_paper_id'));

        if (! $paper->is_published) {
            abort(403, 'This paper is not available.');
        }

        $userAssessmentTypeIds = $user->examGoals()
            ->where('is_active', true)
            ->pluck('assessment_type_id');

        if ($paper->assessment_type_id && ! $userAssessmentTypeIds->contains($paper->assessment_type_id)) {
            abort(403, 'This paper does not match your exam goals.');
        }

        $session = $this->examPrepService->createMockSession($user, $paper);

        return redirect()->route('practice.show', $session);
    }

    public function dailyPlan(Request $request): JsonResponse
    {
        $user = $request->user();
        $goals = $this->examPrepService->getActiveGoals($user);
        $primaryGoal = $goals->first();

        if (! $primaryGoal) {
            return response()->json(['plan' => null]);
        }

        $plan = $this->examPrepService->getDailyPlan($user, $primaryGoal);

        return response()->json(['plan' => $plan]);
    }
}
```
