## Decisions

| Decision | Choice |
|----------|--------|
| ExamGoal migration trigger | Inline on first visit to `/exam-timetable`, flagged via `study_preferences['exam_goals_migrated']` |

## Backend

### StudyPlannerService — 3 New Methods

**`getAvailablePapers(ExamTimetableEntry $entry): Collection`**
- Queries `QuestionPaper::where('assessment_type_id', $entry->assessment_type_id)->published()`
- Optionally filters by `institution_course_id` when set on entry
- Returns paper list with question counts via `withCount('questions')`
- Returns empty collection if entry has no `assessment_type_id`

**`createMockSession(User $user, ExamTimetableEntry $entry, QuestionPaper $paper): PracticeSession`**
- Absorbed from `ExamPrepService::createMockSession()`
- Creates FullMock session with paper's questions ordered by section + sort_order
- Sets `time_limit_seconds` from paper's `duration_minutes`
- Sets `assessment_type_id` and `institution_course_id` from paper

**`getPredictiveScore(PracticeSession $session): ?array`**
- Exact copy from `ExamPrepService::getPredictiveScore()`
- Returns `[percentage, grade, is_passing, pass_threshold, next_grade, points_to_next]`
- Returns null if session has no `assessment_type_id` or no grading scale

### StudyPlannerService — 1 Modified Method

**`getExamSummary()` — add `focus_topics: string[]`**
- If nearest exam has AOC topics: return up to 3 AOC topic titles
- Else: return up to 3 weakest topic titles (not_started or weak status) from nearest exam
- Falls back to empty array if no topics resolved

### ExamTimetableController Changes

**`index()` — ExamGoal migration:**
1. Check `study_preferences['exam_goals_migrated']` — skip if true
2. Fetch active ExamGoals with `exam_date` set
3. For each goal with `institution_course_id`: create ExamTimetableEntry (check duplicates first)
4. Set flag to true on profile
5. Refresh entries query

**`index()` — new props:**
- `topicReadiness: Record<string, TopicReadiness[]>` — per-entry readiness from `getTopicReadiness()`
- `mockPapers: Record<string, MockPaper[]>` — per-entry papers for entries with `assessment_type_id`

**`startMock(Request $request, ExamTimetableEntry $entry): RedirectResponse`**
- Ownership check (`$entry->user_id !== $user->id`)
- Validate `question_paper_id` (uuid, exists, published, matches entry's `assessment_type_id`)
- Call `studyPlannerService->createMockSession()`
- Redirect to `practice.show`

**New route:** `POST /exam-timetable/{entry}/mock` → `startMock`

### DashboardController Changes

- Remove `ExamPrepService` import and usage
- Replace `exam_prep_card` prop with `exam_timetable_card` using `StudyPlannerService::getExamSummary()`
- Works for both `renderTertiary()` and `renderSecondary()` — move to shared section
- Pass `has_active_exams` boolean

### Practice Results Changes

- Add `has_active_exams` boolean prop to results page data
- Add `predictive_score` prop for `full_mock` sessions via `StudyPlannerService::getPredictiveScore()`

## ExamGoal Migration

**Trigger:** First visit to `/exam-timetable` when `study_preferences['exam_goals_migrated']` is not true.

**Per goal:**
- Must have `is_active = true` AND `exam_date` set
- Must have `institution_course_id` (goals without it are skipped — no direct LevelSubject mapping)
- Duplicate check: same user + `institution_course_id` + `exam_date`
- Creates ExamTimetableEntry with: `institution_course_id`, `assessment_type_id`, generated label, `exam_date`

**Does NOT:** delete ExamGoals, migrate goals without dates, create ongoing sync.
