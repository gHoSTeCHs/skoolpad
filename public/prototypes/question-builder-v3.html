B I H2 • ∑ { } 187 words · editing

Cache replacement policies decide which cache line to evict when a new line must be loaded. Common choices include LRU, FIFO, Belady's Optimal (theoretical), LFU, and clock approximations.

Round Robin, in contrast, is a CPU scheduling algorithm — it gives each ready process a fixed time slice in turn. Students often confuse it because both involve a "rotating" mental model and because some N-way set associative cache implementations describe their fill order as round-robin within a set. The trick of this question is that surface similarity.

A virtual address is the address generated by the CPU during program execution; it lives in the process's logical address space. The MMU translates it to a physical address using a page table, mapping virtual page number → physical frame number plus the unchanged page offset.

Stem A 4-way set associative cache has 64 KB capacity, 64-byte lines. Compute the number of sets for a 32-bit address space.
Final answer (canonical render)
256 sets ±0
Worked steps · part-marks
  1. Lines per set = cap / (assoc × line) = 65536 / (4 × 64) = 256
  2. Offset bits log₂(64) = 6
  3. Index bits log₂(256) = 8
  4. Tag bits 32 − 8 − 6 = 18
