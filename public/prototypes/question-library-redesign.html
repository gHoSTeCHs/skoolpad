Q Quick answer
Published

A virtual address is the address generated by the CPU during program execution; it lives in the process's logical address space. The MMU translates it to a physical address using a page table, mapping virtual page number → physical frame number plus the unchanged page offset.

S Standard answer
Draft · 187 words

Virtual addressing decouples program-visible memory from physical RAM. Each process sees a contiguous, private address space starting at zero, while the OS + MMU map those addresses to scattered physical frames at runtime… [continues for ~150 more words with diagram]

Stem A 4-way set associative cache has 64 KB capacity, 64-byte lines. Compute the number of sets and the tag/index/offset bit widths for a 32-bit address space.
Final answer
Value 256
Unit sets
Tolerance ± 0
Worked steps · optional · awarded as part-marks
  1. Lines per set = cap / (assoc × line) = 65536 / (4 × 64) = 256
  2. Offset bits = log₂(64) = 6
  3. Index bits = log₂(256) = 8
  4. Tag bits = 32 − 8 − 6 = 18

Q
Quick explanation
free tier · ~50 words · why is C correct?
Published

Round Robin belongs to CPU scheduling, not memory caching. The other three are cache replacement policies — LRU, FIFO, and Belady's Optimal all decide which cache line to evict on a miss.

S
Standard explanation
paid · ~200 words · context + why distractors are tempting
Draft · 187 words

Cache replacement policies decide which cache line to evict when a new line must be loaded. The most common are LRU, FIFO, Belady's Optimal (theoretical), LFU, and clock approximations of LRU.
